 MTG pour les nonos

Les types de cartes de MTG

🧱 1. Cartes principales (types de base)

Type Description Exemples
Créature Combat, défense, capacités Grizzly Bears, Serra Angel
Éphémère (Instant) Sorts rapides, jouables à tout moment Lightning Bolt, Cancel
Rituel (Sorcery) Sorts puissants, jouables uniquement à ton tour Demonic Tutor, Wrath of God
Enchantement Effets continus sur le champ de bataille Pacifism, Phyrexian Arena
Artefact Objets, machines, souvent neutres (incolores) Sol Ring, Mind Stone
Terrain (Land) Fournissent du mana Forêt, Île, Temple Garden
Planeswalker Alliés puissants avec plusieurs capacités Chandra, Jace, Liliana

🔹 2. Sous-types (liés aux types principaux)

🔹 Créatures

  • Exemple: Wokes, Conspirationnistes, Libéral, Conservateur, Bardes, Mythes et Légendes, personnages culturels etc….

🔹 Terrains

  • Basiques : Forêt, Plaine, Île, Marais, Montagne
  • Non-basiques : Temple Garden, Command Tower, Cavern of Souls
  • Types spéciaux : Terrain neigeux, Terrain légendaire, etc.

🔹 Enchantements

  • Aura : attaché à une créature ou permanent
  • Global : effet qui reste sur le champ de bataille

🔹 Artefacts

  • Équipement : attaché à une créature pour lui donner des bonus
  • Véhicule : peut devenir créature s’il est "piloté"

🔹 Planeswalkers

  • Tous sont légendaires, avec des points de loyauté
  • Chaque planeswalker a entre 2 et 4 capacités

🏷️ 3. Types spéciaux et étiquettes (plus rares)

Type Description Exemple
Tribal Ajoute un type de créature à un sort non-créature Nameless Inversion
Saga Enchantement qui progresse à chaque tour The Eldest Reborn
Conspiration (format Conspiracy) Commence en jeu, modifie les règles Power Play
Attraction (Un-set/Infinity) Cartes amusantes et aléatoires Lifetime Pass Holder
Emblème Effet permanent, créé par un planeswalker Emblème de Jace

👑 4. Supertypes

Ces mots précèdent le type de carte et ajoutent une signification particulière :
Supertype Effet
Légendaire Un seul exemplaire en jeu à la fois
Bascule (Snow) Interagit avec des mécaniques neigeuses
De base Terrain de base (forêt, etc.)
Monde (World) Ancienne mécanique, un seul en jeu à la fois
Historique Regroupe les artefacts, légendaires et sagas
Comment fonctionne une carte

🧩 Structure d'une carte Magic

Voici les composantes clés d'une carte (prenons l'exemple d’une créature) :

┌─────────────────────────────────────┐ │ Nom de la carte │ ← 1. Nom ├─────────────────────────────────────┤ │ Coût de mana (ex. 2G) │ ← 2. Coût de mana ├─────────────────────────────────────┤ │ Type de carte (ex. Créature - Elfe) │ ← 3. Type / Sous-type ├─────────────────────────────────────┤ │ Encadré de texte │ ← 4. Texte de règles (capacités) │ │ │ │ ├─────────────────────────────────────┤ │ Texte d’ambiance (optionnel) │ ← 5. Saveur / ambiance ├─────────────────────────────────────┤ │ Force / Endurance (2/2) │ ← 6. Stats (créature seulement) └─────────────────────────────────────┘

🔍 1. Le nom

  • Unique dans le jeu (à moins d'être intentionnellement réutilisé)
  • Reflète souvent le flair ou l’univers de la carte

🔮 2. Le coût de mana

  • En haut à droite : détermine combien de mana tu dois payer pour la jouer
  • Exemples :
    • 3R = 3 mana quelconque + 1 rouge
    • 2WW = 2 mana quelconque + 2 blancs
  • Couleurs impliquées définissent aussi l’identité de la carte

🧙 3. Type et sous-type

  • Détermine ce que c’est (créature, rituel, terrain, etc.)
  • Exemple :
    • Créature — Zombie Soldat
    • Éphémère
    • Enchantement — Aura
  • Certains effets ciblent uniquement un type donné

📜 4. Texte de règles (effets)

  • Ce que fait la carte.
  • Peut inclure des capacités clés (ex. : vol, piétinement, vigilance, défense talismanique, etc.)
  • Exemple : Quand cette créature arrive sur le champ de bataille, elle inflige 2 blessures à n’importe quelle cible.

🎭 5. Texte d’ambiance (flavor text) (optionnel)

  • Texte en italique, pas d’effet de jeu.
  • Sert à ajouter de la couleur, humour, poésie ou histoire.
  • Exemple : « Il ne frappe pas fort, mais il frappe là où ça fait mal. »

💪 6. Force / Endurance (créatures seulement)

  • En bas à droite, format X/Y
    • X = Force : les dégâts qu’elle inflige
    • Y = Endurance : les dégâts qu’elle peut encaisser avant de mourir
    • Exemple : une 2/2 inflige 2 dégâts et meurt après avoir subi 2 ou + de dégâts

✅ Fonctionnement en jeu

  1. Tu paies le coût de mana (ex. 2R)
  2. Tu joues la carte (créature arrive, sort s’effectue, enchantement reste)
  3. Si c’est un éphémère ou rituel, l’effet se résout puis va au cimetière
  4. Si c’est une créature, elle reste en jeu et peut attaquer/défendre
  5. Les effets de la carte peuvent être passifs ou déclenchés

📌 Exemple complet :

Nom : Gobelin Explosif Coût : 1R Type : Créature — Gobelin Texte : Quand Gobelin Explosif meurt, il inflige 2 blessures à n’importe quelle cible. Force/Endurance : 2/1 Ambiance : Il est né pour mourir. Et faire mal en partant.

Comment gagner à MTG

⚔️ 1. Aggro (agressif)

But : Gagner rapidement avec des créatures rapides et beaucoup de dégâts.

Caractéristiques :
  • Courbe de mana très basse
  • Créatures à bas coût avec hâte, force élevée, ou synergies
  • Peu ou pas de jeu en fin de partie
Exemples :
  • Mono-Red Burn : dégâts directs au joueur
  • White Weenie : petites créatures blanches, souvent avec vigilance
  • Gruul Aggro (Rouge/Vert) : grosses bêtes rapides, piétinement

🛡️ 2. Control (contrôle)

But : Gérer toutes les menaces ennemies, puis gagner tard dans la partie.

Caractéristiques :
  • Contresorts, anti-créatures, pioche de cartes
  • Peu de créatures au départ
  • Gagne avec une menace unique mais difficile à gérer
Exemples :
  • Azorius Control (Bleu/Blanc) : contres + exil
  • Dimir Control (Bleu/Noir) : discard + contrôle de cimetière
  • Esper Control (Bleu/Noir/Blanc) : mélange de toutes les formes de contrôle

⚖️ 3. Midrange

But : Survivre au début, puis dominer avec des menaces solides et polyvalentes.

Caractéristiques :
  • Créatures efficaces avec effets d’arrivée en jeu
  • Combinaison de contrôle léger et de pression constante
  • Bonne adaptabilité
Exemples :
  • Jund Midrange (Vert/Rouge/Noir) : créatures value, gestion, removal
  • Abzan (Vert/Blanc/Noir) : synergies cimetière et tokens

🔧 4. Combo

But : Assembler des cartes spécifiques qui permettent une victoire soudaine, parfois infinie.

Caractéristiques :
  • Moins axé sur les créatures ou le combat classique
  • Recherche de cartes, ramp, protection
  • Peut paraître passif... jusqu’au moment fatal
Exemples :
  • Storm : jouer plusieurs sorts dans un tour pour déclencher une boucle
  • Infinite Mana Combo : avec Pili-Pala, Devoted Druid, etc.
  • Mélira Combo / Kiki-Jiki / Splinter Twin (créatures infinies)

🧟 5. Ramp

But : Générer rapidement beaucoup de mana pour lancer des sorts gigantesques.

Caractéristiques :
  • Générateurs de mana (créatures, artefacts, sorts)
  • Grosse menace dès le tour 4 ou 5
  • Risque de mort rapide contre aggro
Exemples :
  • Mono-Green Ramp : Llanowar Elves, Nykthos
  • Simic Ramp (Bleu/Vert) : ramp + pioche
  • Eldrazi Ramp : grosses créatures incolores dévastatrices

🎲 6. Tempo

But : Gagner du temps tout en infligeant une pression constante.

Caractéristiques :
  • Petites créatures efficaces
  • Contres ou bounce pour ralentir l’adversaire
  • Tactique : tu avances pendant que lui recule
Exemples :
  • Delver (Bleu/Rouge) : créatures volantes + cheap spells
  • Faeries : flash, vol, tricks

🧙 7. Tribal

But : Utiliser une tribu (type de créature) pour créer des synergies fortes.

Caractéristiques :
  • Buffs entre créatures du même type
  • Rapidité ou résilience selon la tribu
Exemples :
  • Elfes (Vert) : ramp et débordement
  • Gobelins (Rouge) : agressivité, effets en chaîne
  • Zombies (Noir) : résilience, cimetière
  • Anges, Vampires, Sorciers, Humains, Ours (même les drôles !)

👻 8. Mill / Discard / Alt-Win

But : Gagner autrement que par les points de vie.

Caractéristiques :
  • Mill : vider la bibliothèque adverse (ex. Ruin Crab)
  • Discard : vider la main de l’adversaire (ex. Thoughtseize)
  • Alt-Win : conditions de victoire spéciales (Felidar Sovereign, Laboratory Maniac, etc.)

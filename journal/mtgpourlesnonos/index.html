 MTG pour les nonos

Vous n'êtes pas seul. Et nous sommes là pour vous.

Vous êtes créatifs, plein de bonnes idées, fan de mèmes, mais ne savez pas par où commencer pour créer des cartes qui se tiennent? Voici notre guide MAGIC THE GATHERING POUR LES NONOS

Comment fonctionne une carte

🧩 Structure d'une carte Magic

Voici les composantes clés d'une carte :

🔍 1. Le nom

  • Unique dans le jeu (à moins d'être intentionnellement réutilisé)
  • Le nom peut être très simple, comme il peut avoir une saveur
    • ex.: "Gérard D." ou "Gérard D., Boomer Keyboard Warrior" sont des noms possibles

💸 2. Le coût de mana

  • Détermine combien de mana doit être payé pour la jouer
  • 2 types de valeur:
    • Chiffre en gris : coût de mana quelconque, peu importe la couleur
    • Pastilles de couleur : 1 mana de la couleur indiquée par pastille
  • Certaines pastilles sont bi/colores. Dans ce cas, l'une ou l'autre des indications peut être payée pour cette pastille
  • Couleurs impliquées définissent aussi l'identité de la carte
    • ex.: une carte coutant (2)🔴 sera de type Rouge. Son coût est de 2 manas quelconques et 1 Mana rouge.
    • ex.2: une carte coutant (3)🔵🔵⚫⚫ sera de type Bleu&Noir. Son coût est de 3 manas quelconques, 2 Bleus et 2 Noirs.

—

  • À l'écrit : les couleurs ont leur propre code Alphabétique pour faciliter la rédaction: Rouge : R — Bleu : U — Blanc : W — Noir : B — Vert : G La carte (3)🔵🔵⚫⚫ se lira donc 3UUBB.
  • Snow Mana : type de Mana dont la provenance est de supertype "Snow". Elle n'a pas de couleur à proprement parlé et peut provenir de toutes les couleurs (Snow-covered Forest, Snow mountain). Son symbole alphabétique est S.

🖼 3. Illustration

  • L'illustration de la carte

🧙 4. Supertype, Type et sous-type

  • Le type de carte détermine le genre d'action qu'elle permet d'entreprendre, où les effets qu'elles ont sur le champ de bataille.

a. Le Supertype

  • C'est une indication sur la particularité du type de carte.
  • Les supertypes communs sont "Basic", "Legendary" et "Token"

b. Le Type

  • C'est la nature de la carte. Les types sont toujours les mêmes
    • Land (Terrain)
    • Creature (Créature)
    • Artifact (Artéfact)
    • Sorcery (Rituel)
    • Enchantment (Enchantement)
    • Instant (Éphémère)
  • Le type correspond à une mécanique de jeu pour la carte.
  • Une carte peut être de plusieurs types en même temps
    • ex.: Créature Artéfact

c. Le Sous-Type

  • C'est une indication sur le type qui ajoute à la nature. On peut y trouver la race de la créature, le type d'artefact, etc.
    • ex.: Human Wizard

📜 5. Rule / Texte de règles (effets)

  • Ce que fait la carte.
  • C'est l'endroit où se trouveront les mots-clés (keyword abilities), une terminologie utilisées à travers tout le jeu pour définir certains principes communs aux diverses cartes.
  • Elle indique clairement les différentes particularités dans les actions liées à la carte, et ce, dans tous les aspects du jeu.

🎭 6. Flavor / Texte d'ambiance (optionnel)

  • Texte en italique, pas d'effet de jeu.
  • Sert à ajouter de la couleur, humour, poésie ou histoire

💪 7. Force / Endurance (créatures seulement)

  • En bas à droite, format X/Y
    • X = Force : les dégâts qu'elle inflige
    • Y = Endurance : les dégâts qu'elle peut encaisser avant de mourir
  • Exemple : une 2/2 inflige 2 dégâts et meurt après avoir subi 2 ou + de dégâts
Les types de cartes de MTG
  • Creature : permanents (qui restent en jeu) qui attaquent et bloquent.
  • Sorcery : sorts "lents" à effet immédiat.
  • Instant : sorts "rapides" jouables à tout moment.
  • Artifact : objets magiques qui peuvent être aussi Creatures.
  • Enchantment : effets continus ou améliorations
  • Planeswalker : alliés avec abilités liés à leur loyauté
  • Land : sources de mana.

🧌 Creature

  • Rôle : Les Creatures servent à attaquer, bloquer et défendre le joueur.
  • Mécaniques clés :
    • Elles ont généralement une power/toughness (exemple : 3/3).
    • Elles souffrent du summoning sickness (elles ne peuvent pas attaquer ni utiliser des abilities avec le symbole de "tap" le tour où elles arrivent en jeu, sauf exception)
    • Leurs caractéristiques sont généralement liés à certains mots-clés (keyword abilities)
      • Deathtouch : Toute quantité de dégâts infligée par cette créature détruit l'adversaire (mais pas les joueurs/planswalkers)
      • Defender : Ne peut pas attaquer
      • Double Strike : Inflige deux fois ses dégâts en combat (First Strike + Second Strike)
      • First Strike : Inflige les dégâts en premier, avant les autres créatures .
      • Flying : Peut bloquer ou être bloquée uniquement par des creatures avec Flying ou Reach
      • Haste : Peut attaquer ou utiliser ses capacités de tap dès qu'elle arrive en jeu (ignore le summoning sickness)
      • Hexproof : Ne peut pas être ciblée par des sorts ou capacités ennemies
      • Indestructible : Ne peut pas être détruite par des dégâts ou effets « destroy » ; mais d'autres effets (comme réduire sa force à 0) peuvent tout de même la mettre au cimetière
      • Lifelink : Les dégâts infligés par cette creature vous font gagner autant de points de vie
      • Menace : Ne peut être bloquée que par au moins deux créatures .
      • Prowess : Gagne +1/+1 jusqu'à la fin du tour si vous lancez un sort non-creature
      • Reach : Peut bloquer les créatures volantes .
      • Trample : Peut infliger les dégâts excédentaires au joueur ou au planeswalker attaqué
      • Vigilance : Ne se tape pas lorsqu'elle attaque (reste activable pour bloquer ensuite)
  • Sous-types communs : Elf, Goblin, Zombie, Angel, Dragon, Human, Wizard, etc.

🪄 Sorcery

  • Rôle : Sorts puissants à usage unique qui ne peuvent être joués que pendant votre propre main phase, quand la stack est vide.
  • Mécaniques clés :
    • Effet immédiat puis va au graveyard.
    • Certaines habiletés clés (keyword abilities) sont liées aux sorcery
      • Flashback : Permet de lancer le sort depuis votre graveyard pour un coût alternatif, puis exile le sort.
      • Storm : Quand vous lancez ce sort, vous le copiez une fois pour chaque sort lancé avant lui ce tour-ci.
      • Buyback : Si vous payez un coût additionnel, le sort revient en main après sa résolution.
      • Replicate : Vous pouvez payer un coût supplémentaire plusieurs fois pour copier le sort.
      • Overload : Remplace le texte du sort (souvent "target" par "each") pour un effet global plus puissant, à un coût plus élevé.

⚡ Instant

  • Rôle : Sorts rapides qui peuvent être joués à presque n'importe quel moment, même pendant le tour de l'adversaire.
  • Mécaniques clés :
    • Interaction pendant le combat ou en réponse à d'autres sorts.
    • On retrouve les mêmes Keyword abilities que sur les Sorceries

🧰 Artifact

  • Rôle : Objets magiques souvent "colorless" (dont le coût n'a pas de couleur obligatoire), avec des effets variés. Certains artifacts sont des permanents, d'autres peuvent être des creatures.
  • Mécaniques clés :
    • Se sont des modificateurs, généralement liés aux cartes auxquelles elles sont associées
    • Peuvent donner des bonus, produire du mana, ou avoir des abilities activées.
    • → Artifact Creature : mélange Artifact et Creature.
    • Keyword abilities
      • Equip : Ability des Equipment artifacts qui leur permet de s'attacher à une creature pour un coût donné.
      • Crew : Ability des Vehicle artifacts où vous tapez des creatures pour permettre au véhicule de devenir une creature artifact jusqu'à la fin du tour.
      • Modular : Permet à l'artifact creature d'entrer en jeu avec des marqueurs +1/+1 et de transférer ces marqueurs à une autre creature artifact si elle meurt.
      • Imprint : Ability permettant d'exiler une carte sous cet artifact et de produire un effet spécial en fonction de cette carte exilée.
  • Sous-types communs : Equipment, Vehicle, Treasure, Clue, Food.

🔮 Enchantment

  • Rôle : Effets continus ou améliorations durables.
  • Mécaniques clés :
    • Restent en jeu après leur résolution.
    • Souvent divisés en :
      • Auras : enchantent un autre permanent (exemple : Enchant creature).
      • Global enchantments : effets globaux qui n'enchantent pas un permanent précis.
    • Keyword abilities
      • Enchant : Utilisé par les Auras, indique quel type de permanent ou de cible elles peuvent enchanter (ex. Enchant creature).
      • Totem armor : Si la creature enchantée devait être détruite, on retire à la place l'Aura avec Totem armor et on préserve la creature.
      • Bestow : Permet de lancer la carte comme creature ou comme Aura qui donne des bonus à une autre creature.
  • Sous-types communs : Aura, Saga, Curse, Shrine.

🧙 Planeswalker

  • Rôle : Allié puissant avec des abilities activées une fois par tour.
  • Mécaniques clés :
    • Entre avec un nombre de loyalty counters.
    • Possède des abilities qui ajoutent ou retirent aux loyalty counters.
    • Peut être attaqué directement par les creatures adverses.

⛰️Land

  • Rôle : Source principale de mana.
  • Mécaniques clés :
    • Ne coûte pas de mana pour être joué.
    • Limité à un land joué par tour.
    • Souvent ont l'ability tap: Add {mana}.
  • Les Sous-types communs sont liés à une couleur de mana et constituent les Terrains Basiques
    • Plains : Blanc
    • Island : Bleu
    • Swamp : Noir
    • Mountain : Rouge
    • Forest : Vert
  • Il existe d'autres sous-types : Gate, Lair, Desert, etc.

✊Les Supertypes

Les Supertypes sont des qualificatifs qui apparaissent avant le Type de carte (ex : "Legendary Creature"). Ils ajoutent des règles ou des propriétés spéciales aux cartes, mais ne définissent pas par eux-mêmes le rôle de la carte (contrairement aux Types comme Creature, Land, etc.).

Basic

  • Permet d'avoir plusieurs cartes du même nom dans le deck au-delà de la limite habituelle (exemple : Basic Land – Forest). Typiquement utilisé pour les Basic Lands (Plains, Island, Swamp, Mountain, Forest, Wastes).

Legendary

  • Représente des personnages ou objets uniques. Règle spéciale : vous ne pouvez contrôler qu'un seul permanent Legendary d'un même nom sur le champ de bataille à la fois ("Legend rule"). Exemples : Legendary Creature – Human Wizard.

Snow

  • Marqueur indiquant que la carte peut produire ou interagir avec du snow mana ({S}). Utilisé pour des lands, permanents, et parfois des instants/sorceries. Ex : Snow Land – Forest.

🏷️ 3. Types spéciaux et étiquettes (plus rares)

Certaines cartes spéciales ont des mécaniques spécifiques

  • Tribal: Ajoute un type de créature à un sort non-créature
  • Saga: Enchantement qui progresse à chaque tour
  • Conspiration (format Conspiracy): Commence en jeu, modifie les règles
  • Emblème: Effet permanent, créé par un planeswalker
Fonctionnement en jeu

Lors d'un tour, les joueurs

  1. Paient le coût de mana des cartes (ex. 2R)
  2. Jouent la carte (créature arrive, sort s'effectue, enchantement reste)
  3. Si c'est un éphémère ou rituel, l'effet se résout puis va au cimetière
  4. Si c'est une créature, elle reste en jeu pour éventuellement attaquer/défendre
  5. Les effets de la carte peuvent être passifs ou déclenchés
Comment gagner à MTG

⚔️ 1. Aggro (agressif)

But : Gagner rapidement avec des créatures rapides et beaucoup de dégâts.

Caractéristiques :
  • Courbe de mana très basse
  • Créatures à bas coût avec hâte, force élevée, ou synergies
  • Peu ou pas de jeu en fin de partie
Exemples :
  • Mono-Red Burn : dégâts directs au joueur
  • White Weenie : petites créatures blanches, souvent avec vigilance
  • Gruul Aggro (Rouge/Vert) : grosses bêtes rapides, piétinement

🛡️ 2. Control (contrôle)

But : Gérer toutes les menaces ennemies, puis gagner tard dans la partie.

Caractéristiques :
  • Contresorts, anti-créatures, pioche de cartes
  • Peu de créatures au départ
  • Gagne avec une menace unique mais difficile à gérer
Exemples :
  • Azorius Control (Bleu/Blanc) : contres + exil
  • Dimir Control (Bleu/Noir) : discard + contrôle de cimetière
  • Esper Control (Bleu/Noir/Blanc) : mélange de toutes les formes de contrôle

⚖️ 3. Midrange

But : Survivre au début, puis dominer avec des menaces solides et polyvalentes.

Caractéristiques :
  • Créatures efficaces avec effets d'arrivée en jeu
  • Combinaison de contrôle léger et de pression constante
  • Bonne adaptabilité
Exemples :
  • Jund Midrange (Vert/Rouge/Noir) : créatures value, gestion, removal
  • Abzan (Vert/Blanc/Noir) : synergies cimetière et tokens

🔧 4. Combo

But : Assembler des cartes spécifiques qui permettent une victoire soudaine, parfois infinie.

Caractéristiques :
  • Moins axé sur les créatures ou le combat classique
  • Recherche de cartes, ramp, protection
  • Peut paraître passif... jusqu'au moment fatal
Exemples :
  • Storm : jouer plusieurs sorts dans un tour pour déclencher une boucle
  • Infinite Mana Combo : avec Pili-Pala, Devoted Druid, etc.
  • Mélira Combo / Kiki-Jiki / Splinter Twin (créatures infinies)

🧟 5. Ramp

But : Générer rapidement beaucoup de mana pour lancer des sorts gigantesques.

Caractéristiques :
  • Générateurs de mana (créatures, artefacts, sorts)
  • Grosse menace dès le tour 4 ou 5
  • Risque de mort rapide contre aggro
Exemples :
  • Mono-Green Ramp : Llanowar Elves, Nykthos
  • Simic Ramp (Bleu/Vert) : ramp + pioche
  • Eldrazi Ramp : grosses créatures incolores dévastatrices

🎲 6. Tempo

But : Gagner du temps tout en infligeant une pression constante.

Caractéristiques :
  • Petites créatures efficaces
  • Contres ou bounce pour ralentir l'adversaire
  • Tactique : tu avances pendant que lui recule
Exemples :
  • Delver (Bleu/Rouge) : créatures volantes + cheap spells
  • Faeries : flash, vol, tricks

🧙 7. Tribal

But : Utiliser une tribu (type de créature) pour créer des synergies fortes.

Caractéristiques :
  • Buffs entre créatures du même type
  • Rapidité ou résilience selon la tribu
Exemples :
  • Elfes (Vert) : ramp et débordement
  • Gobelins (Rouge) : agressivité, effets en chaîne
  • Zombies (Noir) : résilience, cimetière
  • Anges, Vampires, Sorciers, Humains, Ours (même les drôles !)

👻 8. Mill / Discard / Alt-Win

But : Gagner autrement que par les points de vie.

Caractéristiques :
  • Mill : vider la bibliothèque adverse (ex. Ruin Crab)
  • Discard : vider la main de l'adversaire (ex. Thoughtseize)
  • Alt-Win : conditions de victoire spéciales (Felidar Sovereign, Laboratory Maniac, etc.)
